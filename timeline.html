Reading Guide

Audit & Exam Cards

The top section shows upcoming audits and exams. Click any card to expand it and see which projects are linked to that audit. The left-side color stripe matches the audit's color used throughout the page.

Project Timeline (Gantt Chart)

Each project row shows a horizontal bar representing its duration. The shaded colored bands behind the bars represent audit preparation windows — wider bands mean longer audit periods. Each project also shows shield icons indicating which audits it impacts. Click any row for full details and AI analysis.

Impact Matrix

The bottom table maps projects to audits. A colored circle with an arrow means that project directly impacts that audit. A dash means no connection.
